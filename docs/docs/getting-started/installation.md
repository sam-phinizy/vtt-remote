# Installation

## Requirements

- Foundry VTT v11 or later
- A relay server (self-hosted or use the public instance)

## Installing the Foundry Module

1. In Foundry VTT, go to **Add-on Modules**
2. Click **Install Module**
3. Paste the manifest URL:
   ```
   https://github.com/sam-phinizy/vtt-remote/releases/latest/download/module.json
   ```
4. Click **Install**

## Connecting Your Phone

1. Open your phone's browser
2. Navigate to your relay server URL
3. Scan the QR code shown in Foundry, or enter the room code manually
