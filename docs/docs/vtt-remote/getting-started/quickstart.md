# Quick Start

Get up and running in 5 minutes.

## Step 1: Install the Module

Install the VTT Remote module in Foundry VTT. See [Installation](installation.md) for details.

## Step 2: Enable the Module

1. Launch your Foundry world
2. Go to **Game Settings** â†’ **Manage Modules**
3. Enable **VTT Remote**
4. Save and reload

## Step 3: Connect Your Phone

1. In Foundry, click the VTT Remote button in the sidebar
2. A QR code will appear
3. Scan with your phone's camera
4. You're connected!
